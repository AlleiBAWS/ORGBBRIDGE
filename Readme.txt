# OpenRGB Vanguard Bridge Plugin -- Antonio Allei M -- 2026

Native OpenRGB plugin that mirrors a selected device's color
to a Corsair Vanguard keyboard via HID.

## Requirements
- OpenRGB 0.9+ (Qt 5)
- Qt 5.x MSVC 64-bit
- Visual Studio Build Tools (MSVC)
- hidapi (MSVC build)

## Build
Open project in Qt Creator using:
- Qt 5.x MSVC 64-bit
- Release configuration

## Runtime
Place hidapi.dll next to OpenRGB.exe
